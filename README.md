# Data-Science-and-Machine-Learning

Compilação de materiais úteis ao tratamento de dados e Machine Learning.

Este repositório é construído de forma incremental: à medida que novos projetos, estudos e materiais são desenvolvidos, eles são organizados aqui por categoria.

## 📁 Estrutura

| Pasta | Descrição |
|---|---|
| `classificacao/` | Projetos de classificação supervisionada |
| `regressao/` | Projetos de regressão e previsão de valores contínuos |
| `series-temporais/` | Modelos de previsão em séries temporais |
| `visao-computacional/` | Projetos de detecção e análise de imagem/vídeo |
| `mlops-monitoramento/` | Monitoramento de modelos em produção (drift, PSI) |
| `material-didatico/` | Materiais de ensino e explicações em contexto educacional |

*Novas categorias serão adicionadas conforme o repositório crescer.*

## 📌 Como cada projeto é organizado

Cada subpasta de projeto deve conter, no mínimo:
- Um `README.md` próprio, com objetivo, dataset utilizado, metodologia e principais resultados
- Um `requirements.txt` específico (evitando conflitos entre bibliotecas de projetos diferentes)
